## About The Project

Introducing an application harnessing the power of the Mapon API in conjunction with the Google Maps API to provide an intuitive and informative experience. This app is designed to visually display the trajectory traversed by a car.

## Getting Started


### Installation

<p>1. Clone the repository</p>

```
git clone https://github.com/IlyashenkoA/react-quiz.git
```

<p>2. Install all packages</p>

```
npm install
```

<p>3. Add .env file and keys for: 
<ul>
<li>Mapon API - REACT_APP_MAPON_API</li>
<li>Google Map API - REACT_APP_GOOGLE_API</li>
</ul>
</p>

<p>4. Run the project</p>

```
npm run start
```
